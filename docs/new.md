## **1\. 战略愿景：构建“第二驾驶舱”**

核心隐喻 (Core Metaphor):  
通海南洋不是传统的“出海中介”，而是企业家在不确定时代里的\*\*“第二驾驶舱” (The Second Cockpit)\*\*。  
当客户的主营业务（第一驾驶舱）面临地缘政治或经济波动时，位于新加坡/马来西亚的“第二驾驶舱”提供冗余的控制权、实时的资产感知与合规的避险路径。  
**数字化三大支柱:**

1. **实体化信任:** 通过官网展示全员实名、实体牌照、专家背书，消除“离岸操作”的不安全感。  
2. **业务闭环:** 从流量（官网）-\> 意向（AI工具）-\> 交付（驾驶舱）的全链路数字化。  
3. **数据主权:** 所有数据存储在客户自控的服务器上，不依赖任何第三方 SaaS 云，确保绝对隐私。

## **4\. 智能工具与获客引擎 (Smart Tools)**

### **4.1 AI 智能咨询顾问 (Private AI Agent)**

* **技术:** Dify.ai (私有化部署) \+ Vector DB。  
* **数据源:** 录入新加坡《公司法》、MOM 政策手册、税法指南。  
* **功能:** 7x24小时回答基础合规问题（如“注册资本最低多少？”），并引导留资。  
* **隐私:** 设置系统提示词，严禁回答政治敏感话题，严禁使用公有云训练数据。

### **4.2 资格自测工具 (Lead Magnets)**

* **形式:** Typebot 对话式表单。  
* **银行开户预审:** 输入存款规模与行业 \-\> 输出推荐银行与成功率。  
* **EP 准证自测:** 输入学历与薪资 \-\> 输出通过率预判（红/黄/绿灯）。

## **5\. 客户驾驶舱 (Client Portal) \- 核心交付**

### **5.1 仪表盘 (Dashboard)**

* **欢迎语:** "Good Morning, Mr. Wang. 您的资产全景图。"  
* **隐私模式:** 右上角“眼睛”图标，一键隐藏所有金额数字（变为 \*\*\*\*），便于公共场合查看。  
* **进度轴:** 类似于物流追踪，显示“资料收集 \-\> 合规审核 \-\> 正式递交 \-\> 获批”的实时状态。

### **5.2 专属顾问直连 (Dedicated Advisor)**

* **UI:** 侧边栏底部展示 Ada 或 Andy 的头像。  
* **交互:**  
  * 点击 **WhatsApp** 图标 \-\> 直连个人 WhatsApp。  
  * 点击 **WeChat** 图标 \-\> 弹出个人微信二维码。  
  * 体现“人”的服务，而非冷冰冰的工单系统。

### **5.3 文档保险箱 (Secure Vault)**

* **功能:** 双向文件传输（替代邮件附件）。  
* **安全:** 上传自动 AES-256 加密；预览自动添加水印 "Confidential \- \[User\] \- \[Date\]"。

## **6\. 技术选型架构 (Technical Architecture)**

为了满足 **"部署在自己服务器"** 且 **"使用可靠开源代码"** 的要求，采用以下 Modern Stack：

| 模块 | 选型推荐 | 理由 (Why Open Source?) |
| :---- | :---- | :---- |
| **前端 (Web)** | **Next.js 14** | React 框架，SEO 友好，支持三语言路由，性能极佳。 |
| **后端 (CMS)** | **Strapi v4** | Node.js Headless CMS。完全私有化，数据在自己手中，支持多语言字段。 |
| **数据库** | **PostgreSQL** | 强大的开源关系型数据库，处理复杂业务数据。 |
| **AI 引擎** | **Dify.ai** | 开源 LLM 应用编排平台。支持本地 RAG，无需上传数据到 OpenAI 服务器。 |
| **工作流** | **n8n** | 开源自动化工具（替代 Zapier）。连接表单、CRM、邮件，数据不出内网。 |
| **表单工具** | **Typebot** | 开源对话式表单构建器，体验远超传统表单。 |
| **对象存储** | **MinIO** | AWS S3 的私有化替代品。存储加密的客户文档。 |
| **服务器** | **Docker** | 全容器化部署，易于迁移和备份。 |

## **7\. 高净值 UI 设计规范 (Design System)**

* **色调:** 深海蓝 (Navy Blue \#0A2540) \+ 香槟金 (Champagne Gold \#D4AF37)。  
* **字体:**  
  * CN/TW: Noto Serif SC (宋体，标题) \+ Noto Sans SC (黑体，正文)。  
  * EN: Playfair Display (标题) \+ Inter (正文)。  
* **留白:** 大量使用留白，避免拥挤，营造“私人银行”的高级感。
